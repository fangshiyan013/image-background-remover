## 1. 项目概述

### 1.1 产品定位

一个面向个人用户和小型电商卖家的在线图片背景去除工具，用户上传图片后，AI 自动识别主体并去除背景，输出透明 PNG。

### 1.2 目标用户

- 电商卖家：需要快速处理商品主图
- 设计师 / 自媒体：需要抠图素材
- 普通用户：证件照换底色、头像处理

### 1.3 核心价值

- 无需注册，打开即用
- 处理速度快（< 5 秒）
- 免费额度足够轻度用户日常使用

## 3. 功能需求

### 3.1 核心功能（必须上线）

**F1 - 图片上传**
- 支持点击上传和拖拽上传
- 支持格式：JPG、PNG、WEBP
- 文件大小限制：≤ 10MB
- 上传后立即触发处理，无需额外点击

**F2 - 背景去除**
- 调用 remove.bg API 处理
- 处理过程显示 loading 动画
- 处理失败显示友好错误提示（如额度不足、格式不支持）

**F3 - 结果展示**
- 左右对比展示原图和处理结果
- 结果图背景用棋盘格纹理表示透明区域

**F4 - 下载**
- 一键下载透明 PNG
- 文件名：`removed-bg.png`

**F5 - 重新上传**
- 处理完成后提供"再试一张"按钮，清空当前状态

### 3.2 非功能需求

- 页面首屏加载 < 2 秒
- 移动端响应式适配
- 不存储用户上传的任何图片

## 5. 接口设计

### POST /api/remove-bg

**请求**
- Content-Type: multipart/form-data
- 字段：`image`（File）

**响应**
- 成功：返回 PNG 二进制流，Content-Type: image/png
- 失败：返回 JSON `{ "error": "错误描述" }`，HTTP 4xx/5xx

**错误码**

| HTTP 状态 | 含义 |
| --- | --- |
| 400 | 未上传文件 / 文件过大 |
| 402 | remove.bg 额度不足 |
| 500 | 服务器内部错误 / API Key 未配置 |

## 6. 部署方案

### 6.1 服务器环境

```
Node.js 22+
PM2（进程守护）
Nginx（反向代理 + HTTPS）
```

### 6.2 环境变量

```
REMOVE_BG_API_KEY=your_key_here
```

### 6.3 启动命令

```bash
npm install
npm run build
pm2 start npm --name "bg-remover" -- start
```

### 6.4 Nginx 配置要点

- 反代到 localhost:3000
- 配置 SSL（Let's Encrypt）
- 上传大小限制：`client_max_body_size 15m`

## 7. MVP 范围说明

### 本期做

- 单张图片上传处理
- 基础 UI（上传 + 对比 + 下载）
- 错误处理和友好提示
- 移动端适配

### 本期不做

- 用户注册 / 登录
- 使用量限制 / 付费墙
- 批量处理
- 图片编辑（裁剪、换背景色）
- 历史记录
- 多语言
